Handshake type

The type of the handshake message.

enum {
    hello_request(0), 
    client_hello(1), 
    server_hello(2),
    certificate(11), 
    server_key_exchange (12),
    certificate_request(13), 
    server_hello_done(14),
    certificate_verify(15), 
    client_key_exchange(16),
    finished(20), (255)
} HandshakeType;

TLS 1.2 specification, RFC 5246, p. 37
