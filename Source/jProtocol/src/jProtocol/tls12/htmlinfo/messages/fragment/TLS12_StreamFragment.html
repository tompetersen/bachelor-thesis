Stream fragment

A stream fragment is constructed by using a stream cipher to encrypt the message and an additional MAC (so-called MAC-then-encrypt). An example for a stream cipher is RC4, which is considered weak today, but is contained in TLS 1.2.

Stream fragments are also generated for the null cipher suite TLS_NULL_WITH_NULL_NULL, consisting of the identity operation (i.e., the data is not encrypted, and the MAC size is zero, implying that no MAC is used).

TLS 1.2 specification, RFC 5246, p. 22
