Random

The hello messages include a random structure, which is used later in the protocol.

struct {
    uint32 gmt_unix_time;
    opaque random_bytes[28];
} Random;

gmt_unix_time is the current time and date in standard UNIX 32-bit format and random_bytes an array consisting of 28 bytes generated by a secure random number generator.

TLS 1.2 specification, RFC5246, p. 39
